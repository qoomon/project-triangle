# Project Management Triangle

An interactive, single-page visualization of the classic project management "Iron Triangle" (Fast, Cheap, Good — pick two).

## What It Does

Users click triangle corners or buttons to select two of three constraints. A comic-style speech bubble shows the resulting trade-off. Educational cards explain each dimension.

## Tech

A single self-contained `index.html` — no frameworks, no build step, no dependencies. Just HTML + CSS + vanilla JavaScript.

## Development

Open `index.html` in a browser. That's it.

## Deployment

Deployed to GitHub Pages automatically on push to `main` via `.github/workflows/deploy.yml`.
